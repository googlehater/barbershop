# FADE – Barber Shop

A business card website for a barber shop. Three static pages, built with a focus on clean styling and code structure.

## Tech Stack

- **HTML5** – semantic markup
- **CSS3** – custom styles, flexbox/grid, responsive design

## Project Architecture

```
Fade/
├── src/    # source files
│ ├── css/  # styles
│ ├── html/ # pages (home, services, contacts)
│ └── img/  # images
├── docs/   # documentation (reports, assignment, etc.)
├── README_RUS.md
└── README.md
```

## How to Run

Simply open any `.html` file from the `src/html/` folder in your browser.

## Features

- Pure frontend, no frameworks
- Logical file organization
- Ready to be ported to any backend
