Лабораторная работа 1. Вёрстка + базовая интерактивность. Все лабораторные связаны, выберите себе тему и реализуйте по ней проект. Каждая лабораторная будет надстройкой над прошлой и модифицировать ее.

Цель: освоить HTML, CSS и начальный JavaScript
- Создать минимум 3 разные HTML-страницы, связанные общей навигацией(должна быть возможность перехода между всеми страницами)
- Выполнить адаптивную вёрстку (большие/маленькие экраны. Если сделаете - молодцы, не сделаете - пойму.)
- Реализовать минимум два вида интерактивности на чистом JavaScript. Примеры интерактивности: клиентская валидация формы, показ/скрытие блоков, переключение тёмной/светлой темы, модальное окно, аккордеон, табы, счётчик символов, фильтр элементов. Неважно что вы выберите, главное чтобы был код на JavaScript, реализующий интерактивность(то есть использовать уже готовые компоненты, у которых имеется интерактивность, нельзя)
- Разрешено использовать CSS-фреймворки или утилитарные библиотеки (Bootstrap, Tailwind, Pico CSS и подобные) 
- Проект сдаётся как набор статических файлов (залейте на github и дайте ссылку)
- Оценивается: чистота и читаемость кода, работоспособность js-логики, аккуратный внешний вид

Вопросы по теории:
1. Обязательные теги, которые должны присутствовать в любом валидном HTML5-документе.
2. Как в HTML вставить изображение на страницу? Обязательно ли указывать атрибут alt и зачем он нужен?
3. Как правильно подключить внешний CSS-файл к HTML-документу?
4. Для чего нужны тэги form
5.  Блочная модель элементов CSS
6. Как поменять цвет кнопки при наведении на нее курсора? Псевдоклассы.
7. Flexbox и CSS Grid
8. CSS-селекторы
9. Как с помощью JavaScript получить доступ к элементу
10. Что такое семантические теги и зачем их использовать
